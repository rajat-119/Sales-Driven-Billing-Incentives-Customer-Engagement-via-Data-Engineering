# Sales-Driven-Billing-Incentives-Customer-Engagement-via-Data-Engineering
Objective: Create a scalable data pipeline for processing 15GB/day transactional data from AWS S3 to drive sales incentives and personalize customer engagement.

Techniques: Utilized Python, PySpark for transformative operations, and MySQL for internal databases to ensure data integrity.

Description: Developed a robust data pipeline using Python and PySpark to process transactional data from AWS S3. Enabled personalized customer engagement through incentive mechanisms and generated daily/monthly reports for analysis by feeding processed data into a data mart.
